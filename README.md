# EPF Self-Contribution Calculator

A modern, responsive desktop application built with Python and `customtkinter` to help you calculate, track, and manage your EPF (Employees Provident Fund) self-contributions, basic salary derivation, and personal savings goals.

## Features

- **Income Dashboard**: Easily input your total received salary to derive your basic salary, calculate employer (13%) and employee (11%) EPF contributions, and determine your net salary.
- **Savings Tracker**: Set a monthly saving target percentage to instantly calculate your savings amount and remaining spendable balance.
- **Visualizations**: Interactive pie charts for monthly breakdowns and bar charts to track your income and savings trends over the year.
- **History Management**: Safely store your monthly records locally in JSON format, view them in an organized table, and delete past entries as needed.
- **Dividend Estimator**: Estimate your annual EPF dividends based on your documented self-contributions.
- **Dark/Light Mode**: Full support for system appearance modes.
- **Auto-Updater Check**: Automatically checks GitHub for newer releases so you are always using the latest version.

## Installation & Setup

### Prerequisites
- Python 3.8+
- Required libraries can be installed via `pip`:
```bash
pip install customtkinter matplotlib
```

### Running the App
1. Clone or download this repository.
2. Open a terminal in the project directory.
3. Run the application:
```bash
python app.py
```
